
1) Запустить index.html
2) Нажать кнопку Point
3) Ввести, либо оставить предложенное имя точки
4) Ввести координаты (вводятся в формате X_Y_Z , где '_' это пробел).
5) Повторить операции 2-4 n-раз, для добавления остальных точек
6) Выбрать точки в Scene Collection (нажав на конкретную, точку левой клавишей мыши)
7) После выбора точек (для построения плоскости минимум - 3) при этом отобразятся белым цветом, нажать кнопку Plane
8) Плоскость построена.
